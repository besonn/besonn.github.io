CUDA学习1

2020-04-24 CUDA GPU

写在前面，这一部分主要是CPU、GPU的体系结构和GPU的编程模型，我对这一部分的理解还不是很充分，所以先占个坑，后面的时候再加以补充。

第一个问题：为什么用GPU进行大型计算不用CPU进行呢？

简单来说，CPU中除了有一些计算单元之外，还有许多其他的存储单元，计算单元的数量十分有限。然而GPU中的计算单元非常的多，可以用来加速计算，十分的方便，也很快。

第二个问题：一些基本的软硬件概念。

  1. 线程
    线程可以看作一个问题，一个for循环就可以看成是一个线程。

  2. 线程块
    顾名思义就是许多线程组成的一个模块（blocks）

  3. Kernel （核）
    一个Kernel包含许多的线程块

  4. Shared Memory（SM）
    SM中存放许多的Kernel。

前面3个的属于软件范畴，后面的SM之后的内容都属于硬件的范畴。
