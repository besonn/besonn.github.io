数学建模——模拟退火

2020-07-19

简介

基本思想：主要来源于固体的退火过程，将固体加热然后再缓慢冷却时，从理论上来讲，如果冷却过程足够缓慢，那么冷却中任意一个温度固体都能达到热平衡，而冷却到低温时将达到这一温度下的内能最小状态。该算法就是需要找到这个最小内能的值。

算法的主要步骤

  1. T=T0，生成刚开始退火时的初始温度，随机生成一个初始解x0，并且计算相应的目标函数值E(x0)
  2. 令T等于下一个状态之下的Ti
  3. 根据当前解xi进行扰动，得到一个新解xj，计算相应的E(xj)，得到ΔE=E(xj)−E(xi)\Delta E = E(x_j)-E(x_i)ΔE=E(xj​)−E(xi​)
  4. 若ΔE<0\Delta E < 0ΔE<0，则接受新解xj；否则新解将按e−ΔE/Tie^{-\Delta E/T_i}e−ΔE/Ti​ 接受，Ti是当前温度。
  5. 判断温度Ti下，重复Li次的扰动和接受过程，重复3和4
  6. 判断T是否已经到达目标温度Tj, 是则终止算法，否则继续执行2

问题
背包问题
有N件物品：第i件物品物品值viv_ivi​，重量为wiw_iwi​，vi和wi都是整数。背包容量为W。可以带走哪几样东西使得带走的东西最值钱？

程序

clear
clc
a = 0.95
k = [5;10;13;4;3;11;13;10;8;16;7;4];
k = -k;	% 模拟退火算法是求解最小值，故取负数
d = [2;5;18;3;2;5;10;4;11;7;14;6];
restriction = 46;
num = 12;
sol_new = ones(1,num);         % 生成初始解
E_current = inf;E_best = inf;  
% E_current是当前解对应的目标函数值（即背包中物品总价值）；
% E_new是新解的目标函数值；
% E_best是最优解的
sol_current = sol_new; sol_best = sol_new;
t0=97; tf=3; t=t0;
p=1;

while t>=tf
	for r=1:100
		%产生随机扰动
		tmp=ceil(rand.*num);
		sol_new(1,tmp)=~sol_new(1,tmp);
		
		%检查是否满足约束
		while 1
			q=(sol_new*d <= restriction);
			if ~q
                p=~p;	%实现交错着逆转头尾的第一个1
                tmp=find(sol_new==1);
                if p
                    sol_new(1,tmp)=0;
                else
                    sol_new(1,tmp(end))=0;
                end
            else
                break
			end
		end
		
		% 计算背包中的物品价值
		E_new=sol_new*k;
		if E_new<E_current
            E_current=E_new;
            sol_current=sol_new;
            if E_new<E_best
				% 把冷却过程中最好的解保存下来
                E_best=E_new;
                sol_best=sol_new;
            end
		else
            if rand<exp(-(E_new-E_current)./t)
                E_current=E_new;
                sol_current=sol_new;
            else
                sol_new=sol_current;
            end
		end
	end
	t=t.*a;
end

disp('最优解为：')
sol_best
disp('物品总价值等于：')
val=-E_best;
disp(val)
disp('背包中物品重量是：')
disp(sol_best * d)
