数学建模——线性规划

· 2020-07-15 · # 数学建模

简介

线性规划的目标函数可以是求最大值，也可以是求最小值。为了方便下面给出标准形式。

线性规划的标准形式

min⁡xcTxs.t.Ax≤b\min \limits_{x}c^Tx \quad s.t. \quad Ax \le b xmin​cTxs.t.Ax≤b

一般线性规划的标准型为

min⁡z=∑j=1ncjxj\min z = \sum_{j=1}^{n}c_jx_j minz=j=1∑n​cj​xj​

s.t.∑j=1naijxj≤bj,i=1,2,...,ms.t.\quad \sum_{j=1}^{n}a_{ij}x_{j} \le b_j,\quad i=1,2,...,m s.t.j=1∑n​aij​xj​≤bj​,i=1,2,...,m

非线性规划的Matlab算法

min⁡f(x)s.t.{Ax≤BAeq⋅x=BeqC(x)≤0Ceq(x)=0\min f(x) s.t. \left\{ \begin{aligned} Ax \le B \\ Aeq · x = Beq \\ C(x) \le 0 \\ Ceq(x) = 0 \end{aligned} \right. minf(x)s.t.⎩⎪⎪⎪⎪⎨⎪⎪⎪⎪⎧​Ax≤BAeq⋅x=BeqC(x)≤0Ceq(x)=0​

其中，f(x)是标量函数；A，B，Aeq，Beq是相应维数的矩阵和向量；C(x),Ceq(x)是非线性向量函数。
Matlab中的命令是

X=FMINCON(FUN,X0,A,B,Aeq,Beq,LB,UB,NONLCON,OPTIONS)

它的返回值向量是向量X。其中FUN是.m定义的函数f(x)。X0是x的初始值。A，B，Aeq，Beq定义了线性约束Ax≤BAx \le BAx≤B，$Aeq · x = Beq $，如果没有线性约束，则A=[],B=[],Aeq=[],Beq=[]。LB和UB是变量x的下界和上界，如果上界和下界没有约束，则LB=[],UB=[]; 如果x无下界，则UB=inf。NONLCON是用M文件定义的非线性向量函数C(x),Ceq(x)。OPTIONS定义了优化参数，可以使用MATLAB默认的参数设置。
